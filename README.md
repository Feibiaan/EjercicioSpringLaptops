## Spring Boot

Proyecto Spring con las dependencias / starters:
* H2
* Spring Data JPA
* Spring Web
* Spring Boot Dev Tools

Aplicación API Rest con acceso a base de datos H2 para persistir la información.

El acceso se puede realizar desde Postman o Navegador.
